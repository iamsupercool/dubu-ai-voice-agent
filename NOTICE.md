# 제출물

## GitHub Repository

- 로컬 환경에서 실행 가능한 전체 소스코드
- 공개(public) 또는 리뷰어가 접근 가능한 저장소

## README 필수 포함 항목

저장소의 `README`에는 아래 **5가지 항목을 반드시** 포함해 주세요.

1. **사용 방법** — 로컬에서 프로젝트를 실행하는 방법을 단계별로 안내 (환경 변수 설정, 실행 명령어 등)
2. **주요 기술 스택** — 사용한 언어, 프레임워크, 라이브러리와 선택 이유
3. **구현 범위 및 미구현 사항** — 무엇을 구현했고, 시간 내에 구현하지 못한것은 무엇인지
4. **설계/구현 중 중점을 둔 포인트** — 가장 신경 쓴 부분이나 특별히 고민했던 기술적 결정
5. **추후 고려 사항** — 더 시간이 주어진다면 개선하거나 추가하고 싶은 내용
